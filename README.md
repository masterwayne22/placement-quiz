# 🧠 CS/Tech Quiz App

A modern, high-performance quiz application designed specifically for **placement preparation and technical interview practice**.

Built with a dark, developer-focused aesthetic, the application combines speed, clarity, and gamified engagement to help students test their knowledge across core Computer Science and Technology topics.

---

## ✨ Overview

The CS/Tech Quiz App delivers a distraction-free assessment experience with a strong focus on usability and visual feedback.

The interface uses a **Dark Navy × Electric Blue** design system inspired by modern developer tools and coding environments. Every interaction is optimized for focus, speed, and confidence-building during placement preparation.

The centerpiece of the experience is a **circular animated countdown timer**, providing real-time urgency while maintaining a clean and professional interface.

## 🚀 Key Features

### ⏱ Animated Circular Timer

A visually engaging countdown timer that acts as the primary time indicator for each question.

### 📊 Header-Based Progress System

Instead of using a conventional progress bar, the quiz introduces a unique design decision:

> The progress bar is integrated directly into the header background itself, transforming the entire header into a live progress indicator.

This creates a cleaner layout while continuously communicating quiz completion status.

### 🧩 Placement-Focused Question Set

Questions are designed around common placement and technical interview topics such as:

* Data Structures
* Algorithms
* Operating Systems
* DBMS
* Computer Networks
* OOP Concepts
* Programming Fundamentals
* General Technical Aptitude

### 🎯 Instant Score Tracking

Real-time score updates provide immediate performance feedback.

### 📱 Fully Responsive Layout

Optimized for:

* Desktop
* Laptop
* Tablet
* Mobile Devices

### ⚡ Lightweight & Fast

Built with performance in mind, ensuring smooth interactions and minimal loading overhead.

## 💡 Design Decisions

### Signature Element: Circular Timer

The timer is intentionally positioned as the visual focal point of the application.

Why?

Because placement tests are often time-bound, and making time awareness prominent helps simulate real assessment environments.

---

### Bold UX Experiment: Progress-As-Header

Most quiz applications place progress indicators beneath the header.

This project takes a different approach:

* No standalone progress bar
* No additional visual clutter
* The header itself gradually fills as progress increases

The result is a cleaner and more immersive experience while preserving essential feedback.
